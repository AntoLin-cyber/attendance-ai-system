# 🎓 AI-Powered Attendance Management System

An intelligent web-based system designed to track, analyze, and predict student attendance using **Flask** and **Machine Learning**.

---

## 📌 Overview

This project provides a smart solution for managing student attendance in educational institutions.  
It not only records attendance but also **analyzes trends and predicts future performance** using AI techniques.

---

## 🚀 Features

- 📋 Add and manage student records  
- 🟢 Mark attendance (Present / Absent)  
- 📊 Automatic attendance percentage calculation  
- 🤖 Predict future attendance using Machine Learning  
- 📈 Visualize performance using graphs  
- ⚠️ Identify students below 70% attendance  

---

## 🧠 AI / Machine Learning

This system uses **Linear Regression** to predict attendance trends.

### 🔍 How it works:
- Takes current attendance data  
- Learns pattern based on total working days  
- Predicts future attendance percentage  

### 🎯 Purpose:
Helps identify students at risk before their attendance drops too low.
